Chemical Structure Visualization Web Service

Project Overview

Serverless SMILES Visualization Server

A demonstration of AWS serverless integration featuring:

Frontend

  • Static HTML/JavaScript hosted on GitHub Pages
  • Simple form interface with image display
  • Zero-cost hosting solution

Backend

  • AWS API Gateway with custom domain (projects.vizeet.me)
  • Lambda function for SMILES processing
  • ACM-managed SSL certificates

3. Domain & Certificate Management

sequenceDiagram participant A as ACM participant B as API Gateway participant C as Route53 A->>A: Request Certificate A->>B: Associate with API Gateway B->>C: Create CNAME Record C->>B: Domain Validation

Implementation Steps:

  1. Request public certificate in ACM (us-east-1 region)
  2. Create API Gateway custom domain: 
    Domain name: api.projects.vizeet.me
Base path: /
Stage: production
  3. Create Route53 CNAME record: 
    Type: CNAME
Name: api.projects.vizeet.me
Value: [API Gateway DNS name]
